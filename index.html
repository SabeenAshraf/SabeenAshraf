Projects

A Comprehensive Machine Learning Approach to P2P Lending

Built predictive models to assess loan default risk on P2P lending platforms using the Bondora dataset (2009–2023), addressing data imbalance, outlier detection, and model explainability. Applied extensive preprocessing (capping, encoding, normalization), feature selection (RFECV, feature importance), and evaluated 14 models including Logistic Regression, Decision Tree, KNN, MLP, XGBoost, LightGBM, CatBoost, and AutoGluon. The best-performing model, XGBoost-SMOTE-HYP, achieved AUC of 74.78%, with strong generalization and minimal overfitting. Explainable AI techniques (SHAP, LIME) provided global and local interpretability, identifying key risk factors such as liabilities, education level, and borrower type enhancing transparency and decision-making in loan risk evaluation.

Deep Learning-Based Cancer Detection Using Microscopic Histology Images

Built a breast cancer classification model using the BACH dataset, comprising 400 high-resolution H&E-stained histology images equally distributed across four classes: normal, benign, in situ carcinoma, and invasive carcinoma. Performed manual train-test split, image format conversion, resizing, and custom preprocessing to prepare data for model training. Leveraged transfer learning with ResNet50, InceptionV3, and a custom CNN combined with data augmentation, batch normalization, and dropout layers to prevent overfitting. Achieved a strong AUROC of 93.7%, demonstrating the model's ability to detect cancerous patterns with high accuracy. The project highlights the importance of preprocessing in small medical datasets and showcases the power of deep learning in histopathological image analysis.

Advanced Text Mining for Airbnb Listing Prediction

Built a multilingual text classification pipeline (EN, PT, FR, ES) to predict Airbnb listing unlisting using Natural Language Processing and Machine Learning. Performed extensive text preprocessing and engineered features from property descriptions and host bios (e.g., sentiment, license presence, special symbols). Evaluated models including KNN, Logistic Regression, MLP, LSTM, BERT, and GPT-2 with embeddings like BoW, TF-IDF, and GloVe. Achieved best performance using KNN + BoW + Cosine Similarity (F1-score: 77.9%, Accuracy: 79.6%). Delivered a deployable model and extracted actionable linguistic patterns to support business risk reduction strategies.

Customer Segmentation for Insurance

Performed end-to-end customer segmentation to support targeted marketing strategies. Created a merged clustering solution integrating demographic and premium-based perspectives, identifying three distinct customer profiles. Key variables influencing cluster formation included Premium Motor, Premium Health, and Premium Household. Proposed actionable marketing strategies tailored to each cluster based on age, income, product preference, and loyalty potential. Achieved strong interpretability using decision trees and R² scoring, supporting data-driven decision-making for campaign personalization and cross-selling opportunities.
